# Firestore Release Artifacts

This folder stores immutable snapshots generated by `scripts/firestore_deploy_controlled.ps1`.

Each release gets a folder:

- `docs/deployments/firestore/<releaseVersion>/firestore.rules`
- `docs/deployments/firestore/<releaseVersion>/firestore.indexes.json`
- `docs/deployments/firestore/<releaseVersion>/manifest.json`
- `docs/deployments/firestore/<releaseVersion>/validation-checklist.md`

Suggested release version format:

- `yyyyMMdd-HHmmss` (default generated by script)
- or semantic/team format like `v1.3.0-firestore`
